🧩 Key Components
MapCanvas.tsx
Core WebGL viewport. Uses useRef for map state to avoid re-renders. Renders vector tracks + train markers.
TrainMarker.tsx
Rotated arrow marker showing bearing. Click opens TrainDrawer. Uses LERP position from WebSocket.
WebSocketProvider.tsx
Context provider for real-time train updates. Decodes binary protobuf packets. Falls back to mock data.
TrainDrawer.tsx
Bottom sheet with train details. Sandboxed touch to prevent map pan conflicts.
